# Chillgram Frontend

React приложение для социальной сети Chillgram.

## Технологии
- React 18
- Vite
- React Router
- Socket.io Client
- CSS Modules

## Установка
```bash
npm install
npm run dev
```

## Скрипты
- `npm run dev` - Запуск dev сервера
- `npm run build` - Сборка для продакшена
- `npm run preview` - Предпросмотр сборки
